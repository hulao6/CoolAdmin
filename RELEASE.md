## Quick Start

### Download & Use
```bash
# Download the release
wget https://github.com/your-username/cooladmin/archive/v2.1.0.zip

# Extract and serve
unzip v2.1.0.zip
cd cooladmin-2.1.0
python3 -m http.server 8000

# Open http://localhost:8000
```

### Or Clone
```bash
git clone https://github.com/your-username/cooladmin.git
cd cooladmin
git checkout v2.1.0
```
